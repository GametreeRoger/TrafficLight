# Naming Rules

## Fields

- private field 使用 m_開頭camelCase
- public property 使用 PascalCase
- local variable 使用 camelCase
- const 使用 PascalCase
- private function 使用 camelCase
- public function 使用 PascalCase

```csharp
private int m_countStart;
public int Count;
```

## Properties

Use PascalCase.

```csharp
public int Count { get; set; }
```

## Local Variables

Use camelCase.
